== Description ==

= Client Dash - WordPress.org Plugin Stats is amazing =

Creates a widget which is useable on the dashboard for displaying stats for a specified plugin that is hosted on wordpress.org. It also creates an optional tab on the Reports page for rendering information about plugins hosted on wordpress.org.

== Installation ==

Using this plugin is very simple. All you have to do is:

1. Upload the 'client_dash___wordpress.org_plugin_stats.zip' folder to the '/wp-content/plugins/' directory

2. Activate Client Dash - WordPress.org Plugin Stats through the 'Plugins' menu in WordPress

3. Pat yourself on the back!
